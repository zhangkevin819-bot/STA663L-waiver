Project 1 implements foundational machine learning algorithms including logistic regression, softmax regression, and stochastic gradient descent (SGD), with real-world data applications demonstrating optimization techniques and model convergence.

Project 2 encompasses ensemble learning methods including decision trees, random forests, and gradient boosting machines (GBM), validated through comprehensive simulations on real datasets to demonstrate model performance and comparative analysis.

Project 3 explores reinforcement learning frameworks through implementations of Markov Decision Processes (MDP), Monte Carlo methods, Temporal-Difference (TD) algorithms, Q-Learning, and SARSA, showcasing agent-environment interaction and policy optimization.

Project 4 demonstrates deep learning architectures including multi-layer perceptrons (MLP), convolutional neural networks (CNN), and Vision Transformers (ViT), applied to computer vision tasks with real-world datasets.

The Advanced ML Pipeline project delivers a production-grade machine learning system encompassing: Python engineering best practices (type annotations, decorators, context managers), high-performance data processing with Polars and NumPy vectorization, custom scikit-learn transformers for feature engineering pipelines, PyTorch implementation of modern Transformer architecture with Rotary Position Embeddings (RoPE) and multi-head self-attention, advanced training techniques including mixed precision training with gradient accumulation and cosine annealing learning rate schedules, MLOps infrastructure featuring FastAPI async inference service, Docker containerization, Hydra hierarchical configuration management, MLflow experiment tracking, and comprehensive unit and property-based testing.
